# Sinu Arcade 🎮
**Sinu Arcade** هي منصة ألعاب لامركزية متكاملة مبنية على شبكة **Base**، مصممة خصيصاً لمجتمع السنديكات. توفر المنصة تجربة ألعاب تفاعلية تعتمد على الحرق (Burn mechanism) وتحديات الرهان المباشر (PvP).

## 🕹️ الألعاب المتوفرة:
* **قطار الاغتيال (PvE):** آلية حرق فوري لدعم سعر التوكن.
* **روليت الحظ (Roulette):** عجلة الحظ السنديكالية.
* **حقل الألغام:** تحدي الغنائم والمخاطر.
* **معركة النرد (PvP):** مواجهة الـ 1vs1 الحاسمة.
* **دامة السنديكات (Checkers):** طاولة التحدي الاستراتيجي المستقلة.
* **دومينو المافيا:** نمط التحدي والمنافسة الجماعية.

## 🚀 التقنيات المستخدمة:
- **Blockchain:** Base Network.
- **Frontend:** HTML5, CSS3 (Responsive Design), JavaScript.
- **Web3 Integration:** Ethers.js.

## 🛠️ كيفية التشغيل:
1. تأكد من ربط محفظتك (MetaMask/Trust Wallet) بشبكة **Base**.
2. تأكد من توفر رصيد $SINU للمشاركة في التحديات.
3. اختر اللعبة من القائمة الرئيسية واستمتع بالتجربة.
